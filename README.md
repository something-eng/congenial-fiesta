# RegnumElementorum

A simple HTML layout that displays two knowledge base documents side-by-side.

## Overview

This project uses a modern flexbox layout to render two HTML pages in equal-width frames:

- `KB1.html`
- `KB2.html`

## Features

- Responsive 50/50 split layout using CSS Flexbox
- Each frame fills half of the browser viewport
- Clean, borderless embedded content with `iframe`

## Notes

- The HTML file uses `iframe` elements instead of legacy `frame` tags.
- The browser viewport is constrained to the full available width and height.

## Usage

Open the HTML file in a browser to view the two knowledge bases side-by-side.
